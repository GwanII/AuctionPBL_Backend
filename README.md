# AuctionPBL_Backend
2025 2학기 PBL 준비 
기존 중고거래는 선착순인것과 다르게 경매 시스템을 도입한 중고 거래 애플리캐이션
포인트 결제 -> 포인트로 경매 진행 -> 포인트 현금 변환 과정
경매자는 최소 입찰가와 경매 기간 설정하여 게시글 생성
구매자는 원하는 게시글 들어가서 경매 진행 
구매자에게 개인 채팅 보내는 기능 보유
찜(즐겨찾기) 기능 보유
